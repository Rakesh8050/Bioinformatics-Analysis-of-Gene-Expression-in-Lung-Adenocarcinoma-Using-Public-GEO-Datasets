01 / Project Overview

About This Study

Lung adenocarcinoma (LUAD) is the most common histological subtype of lung cancer, contributing substantially to global cancer mortality. Its molecular heterogeneity and the strong influence of the tumor microenvironment on bulk transcriptomic profiles motivate multi-dataset analyses.

This study integrates two independent count-based GEO RNA-seq datasets (GSE229705 & GSE140343) for model development and reserves a third RPKM-quantified dataset (GSE40419) for strictly independent external validation. The pipeline spans differential expression, functional enrichment, co-expression network analysis, immune cell deconvolution, LASSO feature selection, and four supervised ML classifiers.

🎯

Objective

Identify reproducible transcriptomic biomarkers in LUAD through integrative multi-dataset analysis and machine learning classification.

📊

Scale

346 merged samples · 223 unique patients · 16,282 genes tested across 3 independent GEO datasets.

🔬

Novelty

Patient-level splitting prevents data leakage; cross-quantification-unit external validation reveals calibration requirements for model deployment.

02 / Abstract

Abstract Summary

Background

LUAD's molecular heterogeneity and tumor microenvironment influence on bulk transcriptomics motivate multi-dataset analyses for reproducible biomarker identification with translational potential.

Methods

Two count-based GEO datasets integrated for development; one RPKM dataset reserved for external validation. Pipeline: DESeq2, GO/KEGG enrichment, CEMiTool co-expression, STRING PPI, CIBERSORT deconvolution, LASSO selection, four supervised classifiers.

Results

2,421 significant DEGs (1,249 up, 1,172 down). Hub genes: CD79A, GZMK, JCHAIN, MS4A1, IRF4, CCL19, CXCL13. Best held-out AUC: GBM (0.991). Best external AUC: ANN (0.995). LASSO selected 1,347-gene signature.

Conclusions

Consistent evidence across DEG analysis, co-expression, PPI networks, immune cell analysis, and ML supports robust candidate biomarkers. Calibration-aware deployment is critical across different expression quantification units.

04 / Methodology

Bioinformatics Workflow

01

Data Acquisition & Preprocessing

Three GEO RNA-seq datasets retrieved. Genes retained with ≥10 raw counts in ≥20% samples. Entrez IDs mapped to HGNC symbols. Datasets merged on 16,282 common genes. Patient-level 70/30 split (seed 123) prevents data leakage.

R · GEOquery · edgeR · biomaRt
02

Differential Expression Analysis

DESeq2 on training set only. Log₂FC shrinkage via apeglm method. Significance threshold: adj. p < 0.05 & |log₂FC| ≥ 1. Identified 2,421 significant genes (1,249 up / 1,172 down).

R · DESeq2 · apeglm
03

Functional Enrichment Analysis

Separate GO (BP, MF, CC) and KEGG enrichment for upregulated and downregulated DEGs. Universe: 16,282 tested genes. Benjamini-Hochberg correction. 200 BP terms (up), 400 BP terms (down).

R · clusterProfiler · enrichplot
04

Co-Expression Network Analysis

CEMiTool on VST-transformed, batch-corrected training matrix (limma). Identified 8 modules (M1–M8) plus Not.Correlated. M2/M4 tumor-high; M1/M6/M7 normal-high by eigengene comparison.

R · CEMiTool · limma · VST
05

PPI Network Analysis

STRING database (Homo sapiens, score ≥400) for the immune-high module. Network: 25 nodes, 80 edges. Hub genes by degree, closeness & harmonic centrality via igraph.

R · STRING · igraph
06

Immune Cell Deconvolution

CIBERSORT via IOBR package on CPM-normalized counts (1,000 permutations). Wilcoxon rank-sum + BH correction. Significant shifts in 18 cell types; monocytes most depleted (Δ −0.036).

R · IOBR · CIBERSORT
07

LASSO Feature Selection

L1-regularized logistic regression with internal cross-validation. Optimal C = 3.039. Selected 1,347-gene signature from 2,421 DEGs. Validation AUC = 0.991.

Python · scikit-learn · LogisticRegressionCV
08

Machine Learning Classification

Four classifiers: Random Forest, SVM, GBM, ANN (MLP). Grid search + 5-fold stratified CV optimising ROC AUC. Evaluated by AUC, accuracy, sensitivity, specificity, F1, Brier score on held-out and external sets.

Python · scikit-learn · numpy 2.3.5
